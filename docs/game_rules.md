# Hexor 게임 핵심 규칙 가이드

이 문서는 Hexor 게임의 보드 구성, 매칭 로직, 점수 및 시간 시스템에 대한 핵심 규칙을 정의합니다. 본 규칙을 수정할 때는 반드시 사용자의 허락을 받아야 합니다.

## 1. 보드 및 구성 요소
- **보드 크기**: 7행 6열의 육각형(Hexagon) 타일로 구성됩니다.
- **컬러 바 (Color Bar)**: 화면 상단에 10개의 색상 타일이 흐름(Stream) 형태로 배치됩니다.
- **색상 종류**: Coral, Amber, Mint, Azure, Violet 총 5가지 색상을 사용합니다.

## 2. 매칭 및 드래그 규칙
- **매칭 조건**:
    - 인접한 육각 타일을 드래그하여 연결합니다.
    - 연결된 타일들의 색상 순서가 상단 **컬러 바의 연속된 구간**과 정확히 일치해야 합니다.
    - 최소 매칭 길이는 **3개 타일**입니다.
- **상태 변화**:
    - 매칭에 성공한 구간은 보드에서 사라지고 새로운 타일로 채워집니다.
    - 컬러 바에서 사용된 구간은 제거되고 새로운 색상으로 뒤가 채워집니다.

## 3. 점수 시스템 (Scoring)
- **기본 점수**: 매칭 성공 시 기본 100점이 부여됩니다.
- **길이 보너스**: 3개를 초과하는 타일(`n = 길이 - 3`)에 대해 추가 점수가 부여됩니다.
    - 식: `(n * 150) + (n^2 * 50)`
- **콤보 보너스**: 이전 매칭 후 4초 이내에 다음 매칭을 성공하면 콤보가 쌓입니다.
    - 식: `(콤보 횟수 - 1) * 50`

## 4. 시간 시스템 (Time & Combo)
- **제한 시간**: 기본 60초로 시작합니다.
- **시간 보너스**: 매칭 성공 시 길이에 따라 시간이 추가됩니다. (최대 25초까지 누적 가능)
    - 식: `1.4 + (길이 - 2) * 0.65` 초
- **콤보 판정**: 마지막 매칭 성공 시점으로부터 **4초 이내**에 다음 매칭이 완료되어야 콤보로 인정됩니다.

## 5. 게임 종료 조건
- **시간 초과**: 남은 시간이 0초가 되면 종료됩니다.
- **수 부족 (No Moves)**: 보드 내에서 컬러 바와 일치하는 경로를 더 이상 만들 수 없을 때 종료됩니다. (매칭 성공 후 항상 체크)
